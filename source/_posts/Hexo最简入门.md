# Hexo快速入门

## 安装基本环境
- Node.js

## 下载Hexo
- Hexo本体
```powershell
npm install -g hexo-cli --registry=http://registry.npmmirror.com
```

- 渲染器
```powershell
npm install --save hexo-renderer-less hexo-renderer-ejs --registry=http://registry.npmmirror.com
```

## 主题
- async
```powershell
npm i hexo-theme-async@latest
```
